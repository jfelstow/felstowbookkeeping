FAQ
Do I need to know how to code?
No. The pipeline scripts work as-is after filling in a clearly marked config block. The two files that are custom per client (the spreadsheet math and the dashboard branding) come with copy-paste AI prompts that do the translation for you, plus a verification step so you know the math is right.
What do I need?
A Mac, Python 3, Node.js, and a free GitHub account. Total infrastructure cost: $0/month.
I’m on Windows. Can I use this?
The kit is built and documented on a Mac. The scheduler config in particular is macOS-specific. The Python and Node pieces are cross-platform, but I haven’t tested the full pipeline on Windows. If you’re on Windows, email me first and we’ll figure out whether it fits your setup before you pay.
What exactly do I get, and how is it delivered?
The guides, the production scripts, the Projections template, the full demo dashboard source, and the client-facing materials, delivered as an instant download the moment checkout completes, starting with a Start Here roadmap. Everything is yours to keep and rebrand for your own practice.
How long does setup take?
Plan for an afternoon to go from nothing to your first live dashboard, following the Setup Guide step by step. After that, adding a client is much faster, and Mondays run about 15 minutes per client.
Is this a course?
No. It’s a working system, the same scripts and playbook running for real clients today, with documentation written so you can run it too.
Why is the hosting free if the data is sensitive?
The published file is AES-256 encrypted ciphertext; the password your client enters decrypts it in their browser. Nothing readable ever sits on the host.
Is the scenario / Projections view included?
Yes. The demo dashboard includes the full working Projections view, and the kit ships a compute template (revenue cases, cost modes, owner pay, financing) you adapt per client. The guides position it as your premium pricing tier.
Does it work with QuickBooks?
The system runs from an 18-week cash flow spreadsheet (included structure), which you update from any source: QBO, bank feeds, or the client’s Monday text. The bank balance, not the book balance, drives it: that’s rule #7 in the Blueprint.
What if I get stuck?
Every kit comes with email support from me directly. The Weekly Workflow guide also includes a troubleshooting table for every failure mode I’ve hit running this with real clients.
